A aplicação lista 5 lugares de interesse pessoal meu. Os lugares são posicionados no mapa com marcadores em cada posição, através da API do Google Maps. Contém uma caixa de pesquisa que filtra os lugares de acordo com o texto digitado. Ao selecionar um dos lugares na lista, será mostrado os 5 restaurantes próximos do local seceionado, mostrando seu nome, endereço e telefone


# Instalação
Faça o download do arquivo zip no GitHub para uma pasta local e abra o arquivo index.html em um navegador.
